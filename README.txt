Python Project to create a Coronavirus Web Scraper and Voice Assistant

first scraping covid-19 data from a website,
then build a voice assistant that can answer coronavirus related questions.

parseHub is the tool used to download data from a website.
The URL of the website : https://www.worldometers.info/coronavirus/

install :
    pip install requests
    pip install pywin32
    pip install SpeechRecognition
    pip install pyttsx3
    pip install pyaudio (in case of an error install it from this link "https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio")
